# 26042 식당 입구 대기줄
[26042](https://www.acmicpc.net/problem/26042)
* 분류 : 큐

## 나의 답

```py
import sys
from collections import deque

q = deque()

n = int(input())

max_ = 0 # 길이의 최댓값
end_ = 1e6 # 마지막 학생의 번호
for _ in range(n):
    x = list(map(int, sys.stdin.readline().split()))
    if x[0] == 1:
        q.append(x[1])
    elif x[0] == 2:
        q.popleft()
    # 길이의 최댓값이 갱신될 때    
    if len(q) > max_:
        max_ = len(q)
        end_ = x[1]
    # 길이의 최댓값이 같을 때 end_ 확인
    elif len(q) == max_ and x[1] < end_:
        end_ = x[1]
    
print(max_, end_)
```

* 큐 구현은 했지만 주어진 출력하는 것이 어려웠다
  * 답을 보니 나랑 같은 방법으로 출력값을 저장했다
* 푼 사람이 적어서 다양한 답을 모르겠다..

## 다른 사람 답
* 큐를 안씀
* *b가 뭘까
  * 이후의 값을 list에 담아서 b에 저장
```py
import sys
x=c=l=0
for _ in range(int(input())):
  a,*b=map(int,sys.stdin.readline().split()) # *b
  if a < 2:
    c += 1 # cnt를 씀
    if c > x: x = c; l = b[0] # 최댓값x 갱신
    elif c == x: l = min(l, b[0]) # 최댓값이 같을 때
  else:c -= 1
print(x, l)
```

* 분석해보고 돌려보기
```py
import sys
from collections import deque

input()
queue = deque()
ans = (0, float("inf"))
for line in sys.stdin:
    if line[0] == "1":
        n = int(line.split()[1])
        queue.append(n)
        ans = max(ans, (len(queue), n), key=lambda x: (x[0], -x[1]))
    else:
        queue.popleft()
print(*ans)
```
